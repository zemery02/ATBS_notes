Section Eight: More About Strings

Lesson Nineteen: Advanced String Syntax

  • As you may know, if you were to type a string with an apostrophe, and used single quotes to begin and end the string you would get a syntax error. You can avoid this by beginning and ending the string with double quotes.
  • If you need to use both in your string, then you'll need to use an escape character. An escape character lets you use a charcter that is otherwise impossible to implement into a string.
  • Escape characters begin with the backslash character,"\", followed by a letter associated with the character you'd like to end up with.
  • You can escape a single quote by writing "\'" in your string. Spacing is not affected, so if your using an apostrophe to show ownership, you can put the 's' directly after the single quote just like you normally would. You can check out a full list of escape characters in the section resources guide for this lesson.
  • You can use raw strings to have escape characters appear just as they appear when you type them. Raw strings look exactly the same as a regular string, it just has a lower-case r in the front of the beggining quote. This can be helpful for writing a string with multiple backslashes.
  • You can also use triple quotes(double or single), to use escape characters literally, and you can create a newline character just by hitting the enter key. Python is intelligent enough to know that until you type the closing three quotes.
  • If you were to store a multi-line string(the string type in the bullet above) in a variable, and then type the variable's name into the IDLE window, it would output a string with escape characters wherever you used the enter key or any other type of escape character.
  • You can treat each character in a string just as you would handle an item in a list. If you had a variable called spam and inside it was the string "Hello World!", and you typed spam[0] into the IDLE window, it would output the letter H, because thats the character at the first index of the string. You can also use slices with a string. The in and not in functions also work with a string variable, just remember that this function is case sensitive.

Lesson Twenty: String Methods

  • In this lesson we will cover a few different string methods, mich like lesson 15 where we went over several different list methods.
  • Keep in mind that string methods will return a new string, instead of modifying a string already stored in a variable. This is because strings are immutable.
  • The first 2 methods we'll go over is the upper() and lower() methods. As you might've guessed, the upper() and lower() methods will make every index of the string either upper or lowercase depending on which method you use.
  • Since strings are immutable, using the upper() and lower() methods do not alter the contents of the string variable. If you did want to make a string variable all uppercase or lowercase, you would have to write "variablename = variablename.upper()".
  • These methods can be useful for writing a if statement based on user input. If you wanted to get a yes or no response, you could set th einput variable to all upper or lowercase so that no matter what letters they did or didn't capitalize, it would still recognize the word.
  • The next 2 methods are the isupper() and islower() methods. These return a boolean value based on whether or not the letters in a string are all upper or lowercase. Blank strings return False for both methods. You can strig methods together; so for instance, you coudl write "Hello.upper().isupper()", and it would return True.
  • There are several different string methods that start with "is". There's isalpha(), which returns True if there are only letters in a string; isalnum(), which returns True if the string consists of both letters and numbers; isdecimal() which returns True if a string only consists of numbers; isspace(), which returns True if a string is only whitespace; and finally, istitle() which returns True if the string only consists of titlecase. Titlecase is when you have all of your words' first letter capitalized followed by only lowercase until the next word. Like the upper() and lower() methods, ther is a title() method that capitalizes the first letter of each word in a string.
  • You can also use methods on string indexes. So if you wanted to pick out an index of a string where whitespace existed, and used the isspace() method, it would return True.
  • The next 2 methods are startswith() and endswith(). You can pass in anything from the beginning of a string into the startswith() method and it woudl return True; it could be the first letter, the first word, or even the entirety of the string, so long as you include the first letter of the string. Similarly, the endswith() method returns True so long as you include the last character in the string.
  • The join() method joins together a list of strings you pass through the method, and you can preface the method with a string thats used to separate the items in the list. For example, you could write: ' '.join(['cat', 'bat', 'rat']). the output of this would be a string that contained the words "cat", "bat", and "rat" all separated by a sigle whitespace. You can also use newline characters the separate the items, although it will only appear if you pass the join statement into the print function.
  • The split() method outputs a list where each item is a word in a string that prefaces the ".split()" part of the statement. You can also pass in characters into the split() method to split the characters on a specific character rather than a whitespace. So for example: 'Hello world'.split('o'), would output ['Hell', 'w', 'rld'].
  • The ljust() and rjust() methods will add as many whitespaces and necessary to the left or right of a string to make it the length of the integer passed in. That may have been confusing so here's an example: 'Hello'.rjust(10). This will output ' Hello'; now the length of the string is 10. You can also pass in a second arguement that specifies the fill character so that it isn't just whitespace.
  • There is also a center method that places the text in the center rather than to the left or to the right.
  • The strip(), lstrip(), and rstrip() methods all remove whitespace from a string. Say you stored a string in a variable and used a ljust, rjust, or center method in the declaration, you could use the strip method to take away all the whitespace from that string, or you could just remove all the whitespace from the left or right of the text. This doesn't modify the string variable, it just returns a brand new string.
  • Of course you can also pass in any character(s) you want into the split method to remove that/those character(s).
  • The replace() method takes 2 arguements, what to look for and replace, and what to replace it with. For example, you could write into the IDLE window:
    spam = 'Hello There!'
    spam.replace('There', 'World'), and it would output, a new string that says "Hello World!". Again, this is a new string, but it is not what is stored inside the variable now.
  • The pyperclip module has copy and paste methods. The copy() method takes a string arguement and copies the string onto your computer's clipboard. The paste() method pastes whatever the most recent item from your clipboard was into the IDLE window.
